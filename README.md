# Fatorly Platform Documentation

Bilingual (EN/AR) Mintlify docs for the Fatorly platform.

## Run locally

```bash
npm i -g mint
cd platform-documentation
mint dev
```

## Structure

- `en/` English content, `ar/` Arabic mirror (RTL handled automatically)
- `images/{en,ar}/` screenshots by section
- `snippets/` shared MDX snippets (glossary, callouts)
- `TRACKER.md` progress source of truth

See `../docs/superpowers/plans/2026-06-08-platform-documentation.md` for the build plan
and `../docs/superpowers/specs/2026-06-08-platform-documentation-design.md` for the design.
