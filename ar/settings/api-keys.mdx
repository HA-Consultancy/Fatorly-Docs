تتيح لك مفاتيح API ربط أنظمتك الخاصة (مثل نظام تخطيط الموارد) بواجهة برمجة
التطبيقات في فاتورلي. يمكنك تحديد نطاق كل مفتاح بدقّة وإلغاؤه في أي وقت. افتحها من
**الإعدادات → التكاملات → مفاتيح API**.

## إنشاء مفتاح API

<Steps>
  <Step title="سمِّ المفتاح">
    أدخل **اسمًا** وصفيًا (مثل "تكامل نظام ERP").
  </Step>
  <Step title="اختر النطاقات">
    اختر فقط النطاقات التي يحتاجها التكامل (انظر أدناه).
  </Step>
  <Step title="أنشئ">
    اختر **إنشاء مفتاح API**. يُعرض السر **مرة واحدة فقط** — انسخه فورًا واحفظه في
    مكان آمن. لن تتمكّن من رؤيته مرة أخرى.
  </Step>
</Steps>

## النطاقات

| النطاق | يمنح |
|---|---|
| `invoices:read` / `invoices:write` | قراءة / إنشاء الفواتير. |
| `customers:read` / `customers:write` | قراءة / إدارة العملاء. |
| `vendors:read` / `vendors:write` | قراءة / إدارة الموردين. |
| `items:read` / `items:write` | قراءة / إدارة الأصناف. |
| `participants:read` | البحث عن مشاركي الشبكة. |
| `webhooks:manage` | إدارة اشتراكات الويب هوك. |

## إدارة المفاتيح الموجودة

يعرض الجدول لكل مفتاح **الاسم** (مع بادئة المعرّف) و**النطاقات** و**الحالة**
و**تاريخ الإنشاء** و**آخر استخدام**. اختر **إلغاء** لتعطيل مفتاح فورًا.

<Warning>
يُعرض السر الكامل عند الإنشاء فقط. إذا فُقد مفتاح أو تعرّض للاختراق، **ألغِه** وأنشئ
مفتاحًا جديدًا.
</Warning>
