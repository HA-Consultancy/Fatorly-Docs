هذا القسم مرجعٌ تفاعليٌّ كامل لواجهة **Fatorly Integration API** (الإصدار v1).
تُدرَج كل نقطة نهاية في الشريط الجانبي ضمن **API Reference**، مع مُعاملاتها
ومخطّطات الطلب والاستجابة، إضافةً إلى **ساحة تجريب** داخل الصفحة لإرسال طلبات
حيّة ومشاهدة الاستجابات الفعلية.

<Note>
هل أنت جديد على الواجهة؟ ابدأ بأدلة [نظرة عامة](/ar/developers/overview)
و[المصادقة](/ar/developers/authentication) و[البدء السريع](/ar/developers/quickstart)،
ثم استخدم هذا المرجع للاطّلاع على تفاصيل نقاط النهاية كاملةً. صفحات نقاط النهاية
المولّدة متاحة بالإنجليزية فقط.
</Note>

## عنوان الأساس (Base URL)

```
https://integration.fatorly.com
```

جميع نقاط النهاية مُصدَّرة تحت `/v1` (مثل `POST /v1/invoices`).

## المصادقة

يتطلّب كل طلب مفتاح API الخاص بك في ترويسة `X-Api-Key`.

<Steps>
  <Step title="أنشئ مفتاح API">
    من المنصّة، انتقل إلى **الإعدادات ← مفاتيح API** وأنشئ مفتاحًا. يبدو المفتاح
    بالشكل `fat_…` ويُعرَض **مرّة واحدة فقط** — انسخه فورًا. راجع
    [مفاتيح API](/ar/settings/api-keys).
  </Step>
  <Step title="أضِفه إلى ساحة التجريب">
    في أي صفحة نقطة نهاية أدناه، افتح لوحة **Authorization** والصق مفتاحك في حقل
    **`X-Api-Key`**.
  </Step>
  <Step title="أرسل طلبًا">
    عبّئ المُعاملات واختر **Send** لاستدعاء واجهة Integration API الحيّة.
  </Step>
</Steps>

<Warning>
عامِل مفتاح API كأنّه كلمة مرور. تعمل الاستدعاءات على بيانات شركتك الفعلية. إذا
فُقد المفتاح أو تسرّب، فألغِه من **الإعدادات ← مفاتيح API** وأنشئ مفتاحًا جديدًا.
</Warning>
