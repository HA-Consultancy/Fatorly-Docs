يوفّر فاتورلي مرجعًا تفاعليًا لواجهة برمجة التطبيقات (Integration API) مدعومًا بـ
Scalar. يتيح لك تصفّح كل نقطة نهاية وتنفيذ طلبات حيّة عبر "Try it". المرجع مدمج في
المنصة ومتاح فقط للمستخدمين المسجّلين دخولهم.

<Warning>
يجب أن تكون مسجّلًا الدخول إلى حسابك في فاتورلي لفتح مرجع الواجهة. إذا لم تكن مسجّلًا
الدخول، فسيتم تحويلك إلى صفحة تسجيل الدخول أولًا.
</Warning>

## كيفية المصادقة وتجربة طلب

يتطلّب كل طلب إلى واجهة برمجة التطبيقات مفتاح API يُرسَل في ترويسة `X-Api-Key`.

<Steps>
  <Step title="أنشئ مفتاح API">
    في المنصة، انتقل إلى **الإعدادات ← التكاملات ← مفاتيح API** وأنشئ مفتاحًا. يبدو
    المفتاح بالشكل `fat_…` ويُعرض **مرة واحدة فقط** عند الإنشاء — انسخه فورًا. راجع
    [مفاتيح API](/ar/settings/api-keys) لمزيد من التفاصيل.
  </Step>
  <Step title="أضِف مفتاحك في Scalar">
    في صفحة مرجع الواجهة، افتح لوحة **المصادقة (Authentication)** في Scalar والصق
    مفتاحك في حقل **`X-Api-Key`**.
  </Step>
  <Step title="جرّب طلبًا">
    اختر نقطة نهاية واختر **Try it** لاستدعاء واجهة برمجة التطبيقات الحيّة. يرسل كل
    طلب مفتاحك في ترويسة `X-Api-Key`.
  </Step>
</Steps>

<Note>
تعامل مع مفتاح API كأنه كلمة مرور. إذا فُقد أو تعرّض للاختراق، ألغِه من **الإعدادات ←
التكاملات ← مفاتيح API** وأنشئ مفتاحًا جديدًا.
</Note>
