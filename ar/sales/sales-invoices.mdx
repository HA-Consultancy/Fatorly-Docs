تعرض صفحة فواتير المبيعات الفواتير الصادرة التي تصدرها لعملائك — الفواتير الضريبية
والفواتير التجارية معًا.

<Frame>
  <img src="/images/ar/sales/sales-invoices-1.png" alt="قائمة فواتير المبيعات في فاتورلي" />
</Frame>

## القائمة

يعرض كل صف **رقم الفاتورة** و**تاريخ الإصدار** و**المشتري** و**الفئة** و**الإجمالي**
و**الضريبة** و**الحالة**. استخدم **البحث** للتصفية وعناوين الأعمدة للترتيب. إجراءات
الصف هي **عرض** و(للمسودات) **حذف**.

## إنشاء فاتورة

اختر **فاتورة ضريبية جديدة** أو **فاتورة تجارية جديدة** لفتح
[محرّر الفواتير](/ar/invoicing/create-invoice).

- **فاتورة ضريبية** — للمشترين المسجّلين ضريبيًا.
- **فاتورة تجارية** — للمستهلكين أو التصدير.

## الحالات

| الحالة | المعنى |
|---|---|
| مسودة | محفوظة لكن لم تُرسل بعد. |
| متحقّق منها | اجتازت التحقق وجاهزة للإرسال. |
| مُرسلة / مُسلّمة | أُرسلت عبر PEPPOL. |
| فشل الإرسال | فشل الإرسال — افتح الفاتورة لمعرفة السبب. |
| مُستلمة | مستند وارد. |

## التصدير

اختر **تصدير CSV** لتنزيل القائمة كجدول بيانات.

<Tip>
اختر **رقم الفاتورة** أو **عرض** لفتح المستند، حيث يمكنك الطباعة أو التكرار أو
التعديل أو [الإرسال](/ar/invoicing/view-invoice).
</Tip>
