If you cannot remember your password, use this page to request a reset link.
Fatorly emails you a secure link to set a new password.

<Frame>
  <img src="/images/en/getting-started/forgot-password-1.png" alt="Fatorly forgot password page" />
</Frame>

## Request a reset link

<Steps>
  <Step title="Open the page">
    From the [login page](/en/getting-started/login), select **Forgot your
    password?**, or navigate to `/forgot-password`.
  </Step>
  <Step title="Enter your email">
    Type the email address registered to your account.
  </Step>
  <Step title="Complete the security check">
    If a CAPTCHA is shown, complete it.
  </Step>
  <Step title="Send the request">
    Select **Send Reset Link**. A confirmation appears letting you know an email
    is on its way if the address matches an account.
  </Step>
  <Step title="Open the email">
    Follow the link in the email to set a new password — see
    [Reset Password](/en/getting-started/reset-password).
  </Step>
</Steps>

<Note>
For security, the confirmation message is the same whether or not the email
matches an existing account. This prevents others from discovering which emails
are registered.
</Note>

<Tip>
The reset link expires after a limited time. If it stops working, simply request
a new one from this page.
</Tip>

Remembered your password? Select **Sign in** to return to the
[login page](/en/getting-started/login).
