**Downloads** collects every export you generate from a report. Open it from
**Reports → Downloads** in the sidebar, at `/reports/downloads`.

<Frame>
  <img src="/images/en/reports/downloads-1.png" alt="The Fatorly report Downloads list" />
</Frame>

## The list

Each row is one generated export — the file Fatorly built when you ran a report
or when a [scheduled report](/en/reports/scheduled-reports) ran. A row lets you
see which report the file came from, the format it was generated in (CSV, Excel,
or PDF), and when it was created.

## Download a file

Select the download action on a row to save the file to your device. Files
generated from your reports and from your schedules both appear here, so this is
the single place to pick up any export.

<Note>
Generated exports are produced on demand and kept here for convenience. Download
anything you need to retain, and keep your own copy of files you rely on
long-term.
</Note>
